## 1. Описание интеграции

В рамках лабораторной работы была реализована интеграция Telegram-бота с локальным источником данных — CSV-файлом. Бот получает команды пользователя и в зависимости от команды обращается к данным, фильтрует их и возвращает результат.

## 2. Какой источник данных выбрали

В качестве источника данных был выбран CSV-файл `employees.csv`, содержащий информацию о сотрудниках.

---

## 3. Почему именно этот

CSV-файл выбран, так как он прост в использовании, не требует настройки внешних сервисов и легко читается с помощью Python.

---

## 4. Структура данных

Файл employees.csv содержит следующие поля:
- name — имя сотрудника  
- department — отдел  
- position — должность  

Пример строки:
Иван Иванов, IT, developer

## 10. Как работает интеграция

## 11. Ключевые фрагменты кода

Чтение CSV:
```python
df = pd.read_csv("employees.csv")

Поиск по отделу:

df[df["department"] == department]

Поиск по имени:

df[df["name"].str.contains(name, case=False)]
```
